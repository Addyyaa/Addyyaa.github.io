日语活用规则速查表（随身版）

快速查阅版 - 最常用的活用变化规则

🎯 一分钟判断动词类型​

动词词尾 → 类型
───────────────────────────
不是「る」结尾 → 【一类动词】
  例：書く、飲む、話す

「来る」或「する」 → 【三类动词】

「る」结尾：
  ├─ 前面是「い段/え段」→ 通常是【二类动词】
  │   └─ 例：食べる、見る、起きる
  │
  ├─ 前面是「い段/え段」但例外 → 【一类动词】
  │   └─ 例：帰る、切る、入る（需特别记忆）
  │
  └─ 前面不是「い段/え段」→ 【一类动词】
      └─ 例：走る、滑る、知る、減る

⚡ 动词て形变化（30秒记住）​

词尾变化例子
う、つ、るって買う→買って、立つ→立って、帰る→帰って
む、ぶ、ぬんで飲む→飲んで、遊ぶ→遊んで、死ぬ→死んで
くいて書く→書いて
ぐいで泳ぐ→泳いで
すして話す→話して

💡 记忆口诀：うつるって、むぶぬんで、くいて、すして

📝 动词た形变化（和て形一样！）​

规则：把て形中的「て」→「た」、「で」→「だ」

て形た形
書いて書いた
読んで読んだ
話して話した

🔥 动词ない形（否定）​

类型变化例子
一类词尾变「あ段」+ ない書く→書かない、飲む→飲まない
二类去「る」+ ない食べる→食べない、見る→見ない
三类来る→来ない、する→しない

⭐ い形容词变化（必背）​

原形否定过去过去否定副词
高い高くない高かった高くなかった高く

🎯 核心变化：

  • 否定：い → くない
  • 过去：い → かった
  • 副词：い → く

🌟 な形容词变化（必背）​

原形否定过去修饰名词
静か静かではない静かだった静かな

🎯 核心变化：

  • 否定：+ ではない（じゃない）
  • 过去：+ だった
  • 修饰名词：+ な

⚠️ 最容易混淆的词​

一类动词（容易被误认为二类）​

「る」前面是「い段」或「え段」，但实际是一类动词（需特别记忆）：

帰る（かえる）- 回家（「え」在え段，但是一类动词）
切る（きる）- 切（「き」在い段，但是一类动词）
入る（はいる）- 进入（「い」在い段，但是一类动词）
滑る（すべる）- 滑（「べ」在え段，但是一类动词）
減る（へる）- 减少（「へ」在え段，但是一类动词）

其他常见一类动词「る」结尾：

走る（はしる）- 跑
知る（しる）- 知道

な形容词（容易被误认为い形容词）​

きれい（綺麗）- 漂亮 → きれいな
嫌い（きらい）- 讨厌 → 嫌いな

🚀 常用连用规则​

动词 + たい（想...）​

  • 动词ます形（去掉ます）+ たい
  • 食べる → 食べたい（想吃）

动词 + やすい/にくい（容易/难）​

  • 动词ます形（去掉ます）+ やすい/にくい
  • 書く → 書きやすい（容易写）

动词 + ながら（一边...一边）​

  • 动词ます形（去掉ます）+ ながら
  • 食べる → 食べながら（一边吃）

动词て形 + 常用搭配​

  • て + ください（请...）
  • て + いる（正在...）
  • て + もいい（可以...）
  • て + はいけない（不可以...）

📊 完整活用表（快速参考）​

一类动词（書く）​

活用形变化
原形書く
ます形書きます
ない形書かない
て形書いて
た形書いた
ば形書けば
命令形書け

二类动词（食べる）​

活用形变化
原形食べる
ます形食べます
ない形食べない
て形食べて
た形食べた
ば形食べれば
命令形食べろ

三类动词​

活用形する来る
原形する来る
ます形します来ます
ない形しない来ない
て形して来て
た形した来た
ば形すれば来れば
命令形しろ来い

💪 每日练习清单​

N5级别（第1-4周）​

  • 10个动词的ます形、ない形、て形、た形
  • 5个い形容词的变化
  • 5个な形容词的变化
  • 5句て形连接（てください、ている）

N4级别（第5-12周）​

  • 5个动词的完整活用形
  • 动词连接（たい、やすい、ながら）
  • 被动形、使役形各3句

N3级别（第13周以后）​

  • 复合动词练习
  • 敬语动词变化
  • 使役被动形

🎓 学习进度追踪​

N5阶段（30天）
├─ 第1周：动词分类 + 基本活用
├─ 第2周：形容词变化
├─ 第3周：て形、た形熟练
└─ 第4周：综合练习

N4阶段（60天）
├─ 第5-8周：所有动词活用形
├─ 第9-10周：被动形、使役形
└─ 第11-12周：连用规则

N3阶段（90天）
├─ 第13-16周：使役被动形
├─ 第17-20周：复合动词
└─ 第21-24周：敬语动词
