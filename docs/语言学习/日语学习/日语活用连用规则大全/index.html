动词活用体系​

一、动词的三大分类（最重要的基础！）​

【N5 必背】三大类动词的识别方法​

1. 一类动词（五段动词）

  • 识别标志：词尾是「う段」假名（う、く、ぐ、す、つ、ぬ、ぶ、む、る）
  • 特点：活用时，词尾会在「あいうえお」五个段之间变化
  • 例子：
    • 書く（かく）- 写
    • 飲む（のむ）- 喝
    • 話す（はなす）- 说
    • 注意：有些「る」结尾的也是一类动词！

2. 二类动词（一段动词）

  • 识别标志：词尾是「る」，且「る」前面的假名是「い段」或「え段」
  • 特点：活用时，「る」直接去掉，变化很简单
  • 例子：
    • 食べる（たべる）- 吃（「べ」是え段）
    • 見る（みる）- 看（「み」是い段）
    • 起きる（おきる）- 起床（「き」是い段）

3. 三类动词（不规则动词）

  • 只有两个：来る（くる）- 来，する - 做
  • 特点：变化完全不按套路，必须单独背

⚠️ 易混淆点：

  • 「帰る（かえる）」- 虽然「え」在え段，但是一类动词！
  • 「切る（きる）」- 虽然「き」在い段，但是一类动词！
  • 记忆技巧：常见的「〜る」动词中，大部分都是一类动词，只有少数是一类

二、动词的六大活用形​

【N4 核心】活用形的实际用途​

1. 未然形（否定/被动/使役）

  • 一类动词：词尾变成「あ段」+ ない
    • 書く → 書かない（不写）
    • 飲む → 飲まない（不喝）
  • 二类动词：去掉「る」+ ない
    • 食べる → 食べない（不吃）
    • 見る → 見ない（不看）
  • 三类动词：
    • 来る → 来ない（不来）
    • する → しない（不做）

2. 连用形（接ます/て/た）

  • 一类动词：词尾变成「い段」
    • 書く → 書き（接ます：書きます）
    • 飲む → 飲み（接ます：飲みます）
    • 特殊变化（て形）：
      • う、つ、る → って
      • む、ぶ、ぬ → んで
      • く → いて（ぐ → いで）
      • す → して
  • 二类动词：去掉「る」直接接
    • 食べる → 食べ（接ます：食べます，接て：食べて）
  • 三类动词：
    • 来る → 来（来ます、来て）
    • する → し（します、して）

3. 终止形（原形，句尾用）

  • 就是动词的字典形式，直接用在句尾
    • 書く、飲む、食べる

4. 连体形（修饰名词）

  • 和终止形一样，放在名词前面修饰
    • 書く本（要写的书）
    • 飲む水（要喝的水）

5. 假定形（如果...）

  • 一类动词：词尾变成「え段」+ ば
    • 書く → 書けば（如果写）
    • 飲む → 飲めば（如果喝）
  • 二类动词：去掉「る」+ れば
    • 食べる → 食べれば（如果吃）
  • 三类动词：
    • 来る → 来れば（如果来）
    • する → すれば（如果做）

6. 命令形（命令语气）

  • 一类动词：词尾变成「え段」
    • 書く → 書け（写！）
    • 飲む → 飲め（喝！）
  • 二类动词：去掉「る」+ ろ/よ
    • 食べる → 食べろ/食べよ（吃！）
  • 三类动词：
    • 来る → 来い（来！）
    • する → しろ/せよ（做！）

三、动词的时态和体​

【N3 进阶】时态的变化规则​

现在时（原形/ます形）

  • 表示现在的状态或习惯性动作
  • 書く / 書きます（写）

过去时（た形）

  • 变化规则和て形一模一样，只是把「て」换成「た」
  • 書く → 書いた（写了）
  • 飲む → 飲んだ（喝了）
  • 食べる → 食べた（吃了）

进行时（ている）

  • 动词て形 + いる
  • 書いている（正在写）
  • 食べている（正在吃）

完成时（たことがある）

  • 动词た形 + ことがある
  • 書いたことがある（写过）

持续状态（ている）

  • 自动词的ている表示结果持续
  • ドアが開いている（门开着）

四、动词的语态变化​

【N2-N1 进阶】被动、使役、使役被动​

1. 被动形（被...）

  • 一类动词：词尾变成「あ段」+ れる
    • 書く → 書かれる（被写）
    • 飲む → 飲まれる（被喝）
  • 二类动词：去掉「る」+ られる
    • 食べる → 食べられる（被吃）
  • 三类动词：
    • 来る → 来られる（被来）
    • する → される（被做）

2. 使役形（让...做）

  • 一类动词：词尾变成「あ段」+ せる
    • 書く → 書かせる（让写）
    • 飲む → 飲ませる（让喝）
  • 二类动词：去掉「る」+ させる
    • 食べる → 食べさせる（让吃）
  • 三类动词：
    • 来る → 来させる（让来）
    • する → させる（让做）

3. 使役被动形（被迫...）

  • 使役形 + られる
  • 書かせられる（被迫写）
  • 飲ませられる（被迫喝）
  • 口语简化：書かされる、飲まされる

形容词活用体系​

一、形容词的两大分类​

【N5 必背】い形容词 vs な形容词​

1. い形容词（形容词）

  • 识别标志：词尾是「い」
  • 特点：可以直接修饰名词，不需要「な」
  • 例子：
    • 高い（たかい）- 高的
    • 大きい（おおきい）- 大的
    • 美味しい（おいしい）- 好吃的
  • ⚠️ 注意：有些词虽然以「い」结尾，但是「な形容词」！
    • きれい（綺麗）- 漂亮（虽然以い结尾，但是「きれいな」）
    • 嫌い（きらい）- 讨厌（「嫌いな」）

2. な形容词（形容动词）

  • 识别标志：词尾不是「い」，或者是特殊的「い」结尾
  • 特点：修饰名词时必须加「な」
  • 例子：
    • 静か（しずか）- 安静的 → 静かな部屋（安静的房间）
    • 有名（ゆうめい）- 有名的 → 有名な人（有名的人）
    • 好き（すき）- 喜欢的 → 好きな人（喜欢的人）

二、い形容词的活用​

【N4 核心】い形容词的变化规则​

1. 肯定形（现在）

  • 直接使用
  • 高い（高）
  • 美味しい（好吃）

2. 否定形（不...）

  • 把「い」变成「くない」
  • 高い → 高くない（不高）
  • 美味しい → 美味しくない（不好吃）
  • 过去否定：高くなかった（不高）

3. 过去形（...了）

  • 把「い」变成「かった」
  • 高い → 高かった（高了）
  • 美味しい → 美味しかった（好吃）

4. 副词形（...地）

  • 把「い」变成「く」
  • 高い → 高く（高地）
  • 早い → 早く（早地/快地）

5. 名词化（...的）

  • 把「い」变成「さ」
  • 高い → 高さ（高度）
  • 大きい → 大きさ（大小）

6. 连用形（中顿）

  • 把「い」变成「くて」
  • 高くて美しい（又高又美）
  • 安くて美味しい（又便宜又好吃）

三、な形容词的活用​

【N4 核心】な形容词的变化规则​

1. 肯定形（现在）

  • 直接使用，或加「だ」
  • 静か / 静かだ（安静）

2. 否定形（不...）

  • 加「ではない」或「じゃない」
  • 静かではない / 静かじゃない（不安静）
  • 过去否定：静かではなかった（不安静）

3. 过去形（...了）

  • 加「だった」
  • 静かだった（安静）

4. 副词形（...地）

  • 加「に」
  • 静かに（安静地）
  • 上手に（熟练地）

5. 名词化（...的）

  • 加「さ」
  • 静かさ（安静的程度）
  • 便利さ（便利程度）

6. 修饰名词（...的）

  • 加「な」
  • 静かな部屋（安静的房间）

7. 连用形（中顿）

  • 加「で」
  • 静かで綺麗（又安静又漂亮）

名词活用体系​

【N3 进阶】名词的活用变化​

名词本身不变化，主要通过后接助词和助动词来表达不同意思

1. 肯定形

  • 学生だ / 学生です（是学生）

2. 否定形

  • 学生ではない / 学生じゃない（不是学生）
  • 学生ではありません / 学生じゃありません（不是学生 - 敬语）

3. 过去形

  • 学生だった / 学生でした（是学生）

4. 过去否定形

  • 学生ではなかった / 学生じゃなかった（不是学生）

5. 修饰名词（...的）

  • 学生の本（学生的书）
  • 日本の車（日本的车）

6. 作状语（...地）

  • 普通に（普通地）
  • 特別に（特别地）

常见连用规则​

【N4-N3 核心】动词如何连接其他词​

一、动词 + たい（想...）​

  • 规则：动词ます形（去掉ます）+ たい
  • 例子：
    • 食べたい（想吃）
    • 行きたい（想去）
    • 見たい（想看）
  • 注意：这是い形容词的变化，所以否定是「食べたくない」

二、动词 + やすい/にくい（容易/难...）​

  • 规则：动词ます形（去掉ます）+ やすい/にくい
  • 例子：
    • 書きやすい（容易写）
    • 読みにくい（难读）
    • 覚えやすい（容易记住）
  • 注意：这也是い形容词的变化

三、动词 + すぎる（太...了）​

  • 规则：动词ます形（去掉ます）+ すぎる
  • 例子：
    • 食べすぎる（吃太多了）
    • 話しすぎる（说太多了）
  • 注意：すぎる是二类动词，所以过去形是「食べすぎた」

四、动词 + ながら（一边...一边...）​

  • 规则：动词ます形（去掉ます）+ ながら
  • 例子：
    • 食べながら（一边吃）
    • 歩きながら（一边走）
    • 音楽を聞きながら勉強する（一边听音乐一边学习）

五、动词 + て形 + 各种表达​

1. て形 + ください（请...）

  • 書いてください（请写）

2. て形 + いる（正在...）

  • 書いている（正在写）

3. て形 + もいい（可以...）

  • 食べてもいい（可以吃）

4. て形 + はいけない（不可以...）

  • 食べてはいけない（不可以吃）

5. て形 + しまう（...完了）

  • 食べてしまう（吃完了）

易错点与注意事项​

【必看】最容易出错的地方​

1. 动词分类的陷阱​

❌ 错误判断：

  • 「帰る」看起来像二类动词（「え」在え段），但实际是一类动词
  • 「切る」看起来像二类动词（「き」在い段），但实际是一类动词
  • 「入る」看起来像二类动词，但实际是一类动词

✅ 正确记忆法：

  • 只有词尾是「る」，且前面是「い段」或「え段」的，才是二类动词
  • 常见的一类动词「る」结尾：帰る、切る、入る、走る、滑る、知る、減る

2. い形容词 vs な形容词的混淆​

❌ 容易搞错的词：

  • きれい（綺麗）- 虽然以「い」结尾，但是な形容词（きれいな）
  • 嫌い（きらい）- 虽然以「い」结尾，但是な形容词（嫌いな）
  • 大きい（おおきい）- 是い形容词，但对应的「大きな」是连体词

✅ 记忆技巧：

  • 只要记住：きれい、嫌い、きらい是な形容词
  • 其他以「い」结尾的基本都是い形容词

3. て形的特殊变化​

⚠️ 最容易错的地方：

原形て形记忆技巧
書く書いて「く」→「いて」
泳ぐ泳いで「ぐ」→「いで」
話す話して「す」→「して」
立つ立って「つ」→「って」
死ぬ死んで「ぬ」→「んで」
遊ぶ遊んで「ぶ」→「んで」
読む読んで「む」→「んで」
帰る帰って「る」→「って」
買う買って「う」→「って」

🎯 记忆口诀：

  • 「う、つ、る」→「って」
  • 「む、ぶ、ぬ」→「んで」
  • 「く」→「いて」（「ぐ」→「いで」）
  • 「す」→「して」

4. た形和て形的关系​

✅ 重要：た形的变化规则完全一样，只是把「て」换成「た」，「で」换成「だ」

  • 書いて → 書いた
  • 読んで → 読んだ
  • 話して → 話した

5. 二类动词的「られる」​

⚠️ 注意：二类动词的「られる」可以表示：

  • 可能：食べられる（能吃）
  • 被动：食べられる（被吃）

要根据上下文判断！

6. 形容词否定形的过去时​

い形容词：

  • 现在否定：高くない
  • 过去否定：高くなかった（注意是「かっ」不是「く」）

な形容词：

  • 现在否定：静かではない
  • 过去否定：静かではなかった

【N3 进阶】高级活用注意事项​

1. 敬语动词的特殊变化​

尊敬语：

  • 食べる → 召し上がる（吃 - 敬语）
  • 来る → いらっしゃる（来 - 敬语）
  • する → なさる（做 - 敬语）

谦让语：

  • 食べる → いただく（吃 - 谦语）
  • 来る → 参る（来 - 谦语）
  • する → いたす（做 - 谦语）

2. 动词的复合形式​

动词 + 动词（复合动词）：

  • 書き始める（开始写）
  • 読み終わる（读完）
  • 走り回る（跑来跑去）

变化规则：前面的动词用ます形（去掉ます）

3. 动词的名词化​

方法一：动词ます形（去掉ます）直接作名词

  • 読む → 読み（阅读）
  • 食べる → 食べ（吃）

方法二：动词原形 + こと/の

  • 読むこと（阅读这件事）
  • 食べること（吃这件事）

学习建议与记忆技巧​

【学习策略】如何高效掌握活用规则​

1. 循序渐进学习顺序​

第 1 周（N5）：

  • ✅ 掌握动词三大分类
  • ✅ 掌握ます形、ない形、て形、た形
  • ✅ 掌握い形容词和な形容词的区别
  • ✅ 掌握形容词的否定形和过去形

第 2-4 周（N5-N4）：

  • ✅ 掌握所有动词活用形
  • ✅ 掌握形容词的所有变化
  • ✅ 掌握基本连用规则（たい、やすい、ながら）

第 5-12 周（N4-N3）：

  • ✅ 掌握被动形、使役形
  • ✅ 掌握各种て形连接
  • ✅ 掌握复合动词

第 13 周以后（N3-N2）：

  • ✅ 掌握使役被动形
  • ✅ 掌握敬语动词
  • ✅ 掌握高级连用规则

2. 记忆技巧​

技巧一：分类记忆法

  • 把一类动词按词尾分组：う组、つ组、る组、む组、ぶ组、ぬ组
  • 每组一起记て形的变化

技巧二：对比记忆法

  • い形容词和な形容词对比着学
  • て形和た形对比着学

技巧三：造句记忆法

  • 每学一个活用形，马上造 5 个句子
  • 用身边的事物举例

技巧四：口诀记忆法

  • 「うつるって、むぶぬんで、くいて、すして」
  • 「い形容词改い，な形容词加な」

3. 常见错误预防​

错误 1：把一类动词当二类动词

  • 预防：见到「る」结尾的动词，先查字典确认

错误 2：て形变化错误

  • 预防：背熟变化口诀，多练习

错误 3：形容词修饰名词时忘记加「な」

  • 预防：造句子时提醒自己「な形容词必须加な」

错误 4：过去否定形用错

  • 预防：い形容词是「くなかった」，不是「くなかった」

实战练习推荐​

【练习建议】学完立即应用​

每日练习任务​

初级（N5）：

  1. 每天练习 10 个动词的ます形、ない形、て形、た形
  2. 每天造 5 个句子，用い形容词和な形容词修饰名词
  3. 每天练习て形连接（てください、ている）

中级（N4）：

  1. 每天练习 5 个动词的所有活用形
  2. 每天练习动词连接（たい、やすい、ながら）
  3. 每天用被动形和使役形各造 3 个句子

高级（N3-N2）：

  1. 每天练习复合动词
  2. 每天练习敬语动词的变化
  3. 每天用使役被动形造句

动词分类快速判断表​

遇到动词，先看词尾：
├─ 不是「る」结尾 → 一类动词
│   └─ 例：書く、飲む、話す、立つ
│
└─ 是「る」结尾
    ├─ 「る」前面是「い段」或「え段」→ 二类动词
    │   └─ 例：食べる（べ是え段）、見る（み是い段）
    │
    ├─ 是「来る」或「する」→ 三类动词
    │
    └─ 其他「る」结尾 → 一类动词（大部分）
        └─ 例：帰る、切る、入る、走る

て形变化快速判断表​

词尾假名 → 变化结果
─────────────────────
う、つ、る → って
む、ぶ、ぬ → んで
く        → いて
ぐ        → いで
す        → して

形容词快速判断表​

以「い」结尾的词：
├─ 是「きれい」「嫌い」→ な形容词
│
└─ 其他 → い形容词

不以「い」结尾的词：
└─ な形容词

总结​

核心要点回顾​

【最重要】N5 必背 3 条

  1. 动词分三类：一类（五段）、二类（一段）、三类（不规则）
  2. 形容词分两类：い形容词、な形容词
  3. て形变化口诀：「うつるって、むぶぬんで、くいて、すして」

【重要】N4 必背 5 条

  1. 动词六大活用形都要掌握
  2. い形容词变化：い → くない/かった/く
  3. な形容词变化：加ではない/だった/に/な
  4. た形变化和て形一样（て → た、で → だ）
  5. 动词连用：ます形（去掉ます）+ たい/やすい/ながら

【进阶】N3 必背 3 条

  1. 被动形：一类变あ段+れる，二类去る+られる
  2. 使役形：一类变あ段+せる，二类去る+させる
  3. 复合动词：前面动词用ます形（去掉ます）
