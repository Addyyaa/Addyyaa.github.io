# 日语活用规则速查表（随身版）

> 快速查阅版 - 最常用的活用变化规则

---

## 🎯 一分钟判断动词类型

```
动词词尾 → 类型
───────────────────────────
不是「る」结尾 → 【一类动词】
  例：書く、飲む、話す

「来る」或「する」 → 【三类动词】

「る」结尾：
  ├─ 前面是「い段/え段」→ 通常是【二类动词】
  │   └─ 例：食べる、見る、起きる
  │
  ├─ 前面是「い段/え段」但例外 → 【一类动词】
  │   └─ 例：帰る、切る、入る（需特别记忆）
  │
  └─ 前面不是「い段/え段」→ 【一类动词】
      └─ 例：走る、滑る、知る、減る
```

## ⚡ 动词て形变化（30秒记住）

| 词尾 | 变化 | 例子 |
|------|------|------|
| **う、つ、る** | **って** | 買う→買って、立つ→立って、帰る→帰って |
| **む、ぶ、ぬ** | **んで** | 飲む→飲んで、遊ぶ→遊んで、死ぬ→死んで |
| **く** | **いて** | 書く→書いて |
| **ぐ** | **いで** | 泳ぐ→泳いで |
| **す** | **して** | 話す→話して |

**💡 记忆口诀**：うつるって、むぶぬんで、くいて、すして

---

## 📝 动词た形变化（和て形一样！）

**规则**：把て形中的「て」→「た」、「で」→「だ」

| て形 | た形 |
|------|------|
| 書いて | 書いた |
| 読んで | 読んだ |
| 話して | 話した |

---

## 🔥 动词ない形（否定）

| 类型 | 变化 | 例子 |
|------|------|------|
| **一类** | 词尾变「あ段」+ ない | 書く→書かない、飲む→飲まない |
| **二类** | 去「る」+ ない | 食べる→食べない、見る→見ない |
| **三类** | 来る→来ない、する→しない |

---

## ⭐ い形容词变化（必背）

| 原形 | 否定 | 过去 | 过去否定 | 副词 |
|------|------|------|---------|------|
| 高い | 高くない | 高かった | 高くなかった | 高く |

**🎯 核心变化**：
- 否定：い → くない
- 过去：い → かった
- 副词：い → く

---

## 🌟 な形容词变化（必背）

| 原形 | 否定 | 过去 | 修饰名词 |
|------|------|------|---------|
| 静か | 静かではない | 静かだった | 静かな |

**🎯 核心变化**：
- 否定：+ ではない（じゃない）
- 过去：+ だった
- 修饰名词：+ な

---

## ⚠️ 最容易混淆的词

### 一类动词（容易被误认为二类）

**「る」前面是「い段」或「え段」，但实际是一类动词（需特别记忆）**：
```
帰る（かえる）- 回家（「え」在え段，但是一类动词）
切る（きる）- 切（「き」在い段，但是一类动词）
入る（はいる）- 进入（「い」在い段，但是一类动词）
滑る（すべる）- 滑（「べ」在え段，但是一类动词）
減る（へる）- 减少（「へ」在え段，但是一类动词）
```

**其他常见一类动词「る」结尾**：
```
走る（はしる）- 跑
知る（しる）- 知道
```

### な形容词（容易被误认为い形容词）

```
きれい（綺麗）- 漂亮 → きれいな
嫌い（きらい）- 讨厌 → 嫌いな
```

## 🚀 常用连用规则

### 动词 + たい（想...）
- 动词ます形（去掉ます）+ たい
- 食べる → 食べたい（想吃）

### 动词 + やすい/にくい（容易/难）
- 动词ます形（去掉ます）+ やすい/にくい
- 書く → 書きやすい（容易写）

### 动词 + ながら（一边...一边）
- 动词ます形（去掉ます）+ ながら
- 食べる → 食べながら（一边吃）

### 动词て形 + 常用搭配
- て + ください（请...）
- て + いる（正在...）
- て + もいい（可以...）
- て + はいけない（不可以...）
